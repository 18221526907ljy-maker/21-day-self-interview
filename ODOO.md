# Odoo 教育版接入入口

这个仓库已经加入 MindFit / Odoo 教育版接入资料。

推荐从这里开始：

- [`odoo/README.md`](odoo/README.md)：Odoo 教育版接入总说明
- [`odoo/21-day-tasks.md`](odoo/21-day-tasks.md)：可复制到 Odoo 项目的 21 天任务清单
- [`odoo/mindfit-odoo-workflow.md`](odoo/mindfit-odoo-workflow.md)：MindFit 业务流程映射

## 最简单的接入方式

1. 在 Odoo 打开 `项目` 应用。
2. 新建项目：`MindFit - 21天自我采访`。
3. 在项目描述中粘贴仓库链接：
   `https://github.com/18221526907ljy-maker/21-day-self-interview`
4. 创建阶段：`未开始`、`今日进行中`、`已完成`、`需要 MindBot 跟进`、`需要人工关注`。
5. 按 `odoo/21-day-tasks.md` 创建 21 个任务。

这种接入方式适合 Odoo 教育版免费环境，不需要 Odoo.sh，不需要自定义模块，也不需要账号密码。
