# Odoo 教育版接入说明

这份说明用于把 `21-day-self-interview` 接入 Odoo 教育版免费环境。它不需要 Odoo.sh、不需要安装自定义模块，也不需要把账号密码交给任何人。

## 推荐接入方式

在 Odoo 中把本仓库作为 MindFit 的一个服务流程使用：

1. 打开 Odoo 的 `项目` 应用。
2. 新建项目：`MindFit - 21天自我采访`。
3. 在项目描述中粘贴本仓库链接：
   `https://github.com/18221526907ljy-maker/21-day-self-interview`
4. 创建以下阶段：
   - 未开始
   - 今日进行中
   - 已完成
   - 需要 MindBot 跟进
   - 需要人工关注
5. 按照 [`21-day-tasks.md`](21-day-tasks.md) 创建 21 个任务。
6. 如果启用了 `知识库` 或 `文档` 应用，把 [`mindfit-odoo-workflow.md`](mindfit-odoo-workflow.md) 作为流程说明放进去。

## 在 MindFit 中的角色

`21-day-self-interview` 可以作为 MindFit 的轻心理支持模块：

- 面向用户：每天 3 个问题，形成连续 21 天的自我观察体验。
- 面向 MindBot：提供每日提问、记录和回看逻辑。
- 面向运营人员：在 Odoo 中跟踪用户进度、完成状态和需要关注的反馈。
- 面向课程展示：体现 MindFit 的“身材管理 + 情绪支持 + AI轻陪伴 + 社区互动”定位。

## Odoo 中建议使用的标签

- `看见`
- `理解`
- `选择`
- `MindBot`
- `自我采访`
- `情绪支持`
- `需关注`
- `已完成`

## 安全边界

这个项目是自我反思工具，不是医疗诊断或心理治疗。Odoo 中如出现明显危机信号，建议把任务移动到 `需要人工关注` 阶段，并由真实人员跟进。
